# Ekanshdeep Gupta's Personal Website

This is the source code for my personal website, built with [Jekyll](https://jekyllrb.com/).

## Local Development

### Prerequisites

- Ruby
- Bundler

### Installation

1.  Install dependencies:
    ```sh
    $ bundle install
    ```

### Running Locally

To build the site and start a local server:

```sh
$ bundle exec jekyll serve
```

The site will be available at `http://127.0.0.1:4000/`.
